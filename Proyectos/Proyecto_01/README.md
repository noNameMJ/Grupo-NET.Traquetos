# Proyecto 01
En esta carpeta se encuentrar los _.hdl_ de las compuertas lógicas.

## Objetivo
Construir las siguientes compuertas lógicas:
+ Nand (dada)
+ Not
+ And
+ Or
+ Xor
+ Mux
+ DMux
+ Not16
+ And16
+ Or16
+ Mux16
+ Or8Way
+ Mux4Way16
+ Mux8Way16
+ DMux4Way
+ DMux8Way
Se usa exclusivamente la compuerta primitiva **_Nand_** para construir las demás compuertas.
